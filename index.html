Що таке HTML?

HTML (HyperText Markup Language) - це мова розмітки для створення веб-сторінок. Вона визначає структуру та зміст веб-документів за допомогою системи тегів.

Цікавий факт: Перша версія HTML містила лише 18 тегів. Сьогодні HTML5 підтримує понад 140 елементів!

Основні поняття:

  • Теги - базові будівельні блоки HTML
  • Атрибути - додаткова інформація про елементи
  • Семантика - значення елементів для пошукових систем

Семантичний HTML

<header>

Верхня частина сторінки або розділу

<article>

Самостійний блок контенту

<nav>

Навігаційне меню

<footer>

Нижній колонтитул

Приклад семантичної розмітки:

<!DOCTYPE html>
<html lang="uk">
<head>
    <meta charset="UTF-8">
    <title>Семантична сторінка</title>
</head>
<body>
    <header>
        <h1>Заголовок сайту</h1>
        <nav>
            <ul>
                <li><a href="#">Головна</a></li>
                <li><a href="#">Про нас</a></li>
            </ul>
        </nav>
    </header>
    
    <article>
        <h2>Назва статті</h2>
        <p>Основний вміст...</p>
    </article>
    
    <footer>
        <p>© 2023 Мій сайт</p>
    </footer>
</body>
</html>

Що таке штучний інтелект?

Штучний інтелект (ШІ) - це здатність машин імітувати людський інтелект, виконуючи такі завдання як навчання, вирішення проблем, сприйняття та прийняття рішень.

Цікавий факт: До 2025 року світовий ринок ШІ може досягти $190 млрд!

Основні напрямки:

  • Машинне навчання - алгоритми, що вчаться на даних
  • Комп'ютерний зір - розпізнавання зображень
  • Обробка природної мови - розуміння мови

Автопілот

Персоналізовані пропозиції товарів

Етика ШІ

Основні етичні проблеми:

  • Конфіденційність: Використання персональних даних
  • Біас: Дискримінація в алгоритмах
  • Прозорість: "Чорні скриньки" в прийнятті рішень
  • Автоматизація робочих місць: Вплив на ринок праці
