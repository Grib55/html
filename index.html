Що таке HTML?

HTML (HyperText Markup Language) - це мова розмітки для створення веб-сторінок. Вона визначає структуру та зміст веб-документів за допомогою системи тегів.

Цікавий факт: Перша версія HTML містила лише 18 тегів. Сьогодні HTML5 підтримує понад 140 елементів!

Основні поняття:

  • Теги - базові будівельні блоки HTML
  • Атрибути - додаткова інформація про елементи
  • Семантика - значення елементів для пошукових систем

Семантичний HTML

<header>

Верхня частина сторінки або розділу

<article>

Самостійний блок контенту

<nav>

Навігаційне меню

<footer>

Нижній колонтитул

Приклад семантичної розмітки:

<!DOCTYPE html>
<html lang="uk">
<head>
    <meta charset="UTF-8">
    <title>Семантична сторінка</title>
</head>
<body>
    <header>
        <h1>Заголовок сайту</h1>
        <nav>
            <ul>
                <li><a href="#">Головна</a></li>
                <li><a href="#">Про нас</a></li>
            </ul>
        </nav>
    </header>
    
    <article>
        <h2>Назва статті</h2>
        <p>Основний вміст...</p>
    </article>
    
    <footer>
        <p>Мій сайт</p>
    </footer>
</body>
</html>
